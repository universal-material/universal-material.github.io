Property Attribute Type Default Description variant variant import("D:/Projects/universal-material-web/src/button/icon-button").UmIconButtonVariant 'standard' toggle toggle boolean false When true, the button will toggle between selected and unselected states hasSelectionIcon has-selection-icon boolean false selected selected boolean false Sets the selected state. When false, displays the default icon. When true, displays the selected icon, or the default icon If no slot="selected" icon is provided. ariaLabelSelected aria-label-selected string '' The aria-label of the button when the button is toggleable and selected. disabled disabled boolean false Whether the button is disabled or not. href href string '' The URL that the link button points to. target target string '' Where to display the linked href URL for a link button. Common options include _blank to open in a new tab. type type string 'submit' value value string '' name name string undefined form form HTMLFormElement The <form> element to associate the button with (its form owner). The value of this attribute must be the id of a <form> in the same document. (If this attribute is not set, the button is associated with its ancestor <form> element, if any.) https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button#form
